HƯỚNG DẪN MÀN HÌNH IMPORT THUỐC

- Bước 1 : Nhấp vào Tải File Mẫu bên dưới để tải file excel mẫu về máy

- Bước 2 : Mở file excel vừa download về, thêm thông tin của thuốc vào các cell tương ứng trong file. Sau đó lưu lại file

- Bước 3 : Tiếp theo, nhấp vào mục chọn File trong màn hình import

- Bước 4 : Chọn file excel vừa lưu ở bước 2. Sau đó, nhấn Import File và đợi một chút để cập nhật thuốc mới vào kho
